# Fake News Detection using Machine Learning

A Python-based Machine Learning project that classifies news articles as either **True** or **Fake** using Natural Language Processing (NLP) and a Logistic Regression classifier.

## 📊 Final Results
- **Final Model Accuracy:** **98.65%** 🚀
- **Algorithms Used:** TF-IDF Vectorizer and Logistic Regression
- **Language:** Python (Jupyter Notebook)

---
